# Development Server Workflow

This workflow starts the development server for the project.

## Steps

// turbo
1. **Install dependencies** (if needed)
   ```bash
   npm install
   ```

// turbo
2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **Access the application**
   - The server should now be running
   - Check the terminal output for the local URL (typically http://localhost:3000 or similar)
   - Open the URL in your browser

## Notes
- Make sure all dependencies are installed before starting the dev server
- The server will watch for file changes and hot-reload automatically
- Press Ctrl+C in the terminal to stop the server
